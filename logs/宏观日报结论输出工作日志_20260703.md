# 宏观日报结论输出工作日志

日期：2026-07-03

## 一、工作目标

按照“宏观结果输出”PPT的框架，基于最新TREE和日报底层数据，形成可复核、可追溯、可更新的宏观结论，并最终更新到TREE和PPT中。

最终输出不是简单复述指标，而是要完成以下链条：

最新TREE + 日报底层数据 → 重建辅助判断表 → 第一轮筛选指标 → 面级综合判断 → 第二轮筛选展示指标 → 输出标准结论 → 更新TREE/PPT。

## 二、数据源规则

1. 最新TREE是本期框架和指标池的主来源。
   - 使用TREE主表中已有的面级框架、指标归属、当前值、环比、同比、同比增量、权重和指标判断。
   - TREE决定本期有哪些指标可以参与判断。

2. 日报表是底层复核数据源。
   - 用于复核当前值、变化值、历史分位、趋势位置和序列变化。
   - 典型用途包括估值近三年分位、成交额/两融分位、债券收益率变化、公开市场操作变化等。

3. 旧辅助表不能直接沿用。
   - 旧辅助表只作为字段结构和格式模板。
   - 每次更新都必须根据最新TREE和日报重新生成辅助判断表。

4. 其他旧TREE、旧PPT、旧辅助表不作为判断数据源。
   - 旧PPT只能作为样式参考。
   - 旧结论不能直接迁移，必须重新通过辅助表验证。

## 三、辅助表定位

辅助表不是事后复核表，而是结论生成的判断工作台。

它需要回答三类问题：

1. 指标怎么看。
   - 当前值是什么。
   - 环比、同比、同比增量如何。
   - 是否突破关键阈值。
   - 是否处于历史高位或低位。
   - 是否需要和其他指标联动判断。

2. 指标是否进入第一轮候选池。
   - 第一轮候选池用于面级综合判断。
   - 入选条件包括权重高、边际变化大、突破阈值、历史分位异常、指标影响直接、属于宏观锚定指标。

3. 指标是否进入第二轮展示。
   - 第二轮用于最终PPT展示。
   - 每个面最终只选解释力最强、篇幅可控的1-3组指标。

## 五、第一轮筛选规则

第一轮筛选的目标是判断“这个指标是否足以影响面级结论”。

优先进入第一轮的指标：

1. 权重高的核心指标。
   - 例如社零、固投、地产投资、非农、CPI、PPI、核心PCE、DR007、社融、成交额、两融、PE分位等。

2. 边际变化大的指标。
   - 例如环比大幅回落或明显改善。
   - 例如公开市场操作从大额净回笼转为小额净回笼。

3. 突破关键阈值的指标。
   - PMI是否高于50。
   - DR007是否高于7天逆回购。
   - SOFR是否高于IORB。
   - 估值分位是否处于近三年高位。

4. 需要联动判断的指标。
   - DR007必须结合7天逆回购判断银行间资金松紧。
   - SOFR必须结合IORB判断美元短端资金压力。
   - 债市资金面需要结合资金价格和国债收益率。
   - 股市情绪面不能把北向成交额误写成外资净流入。

5. 低频滞后指标一般作为背景。
   - GDP等季度数据通常不进入第一轮。
   - 只有出现明显异常或边际变化特别大时才进入第一轮。

## 六、面级综合判断规则

面级结论必须由第一轮候选池综合得出，不能由最终展示的单一指标倒推。

判断时需要同时看：

- 权重
- 当前值
- 环比
- 同比
- 同比增量
- 历史分位
- 阈值关系
- 指标之间是否互相印证
- 是否存在反向证据

例子：

1. 股市基本面
   - 社零、固投、地产偏弱，说明内需压力仍在。
   - PMI和新订单回升，说明制造业边际改善。
   - 出口和利润改善，说明结构支撑仍在。
   - 因此不能简单写“偏弱”，更适合写“中性”或“中性偏弱”，具体取决于支撑项强弱。

2. 股市资金面
   - 社融和新增贷款环比改善，但同比仍弱。
   - DR007高于7天逆回购，说明银行间资金价格边际收敛。
   - 但这不等于流动性紧缩。
   - 因此应写“中性偏宽松但边际收敛”，而不是“中性偏紧”。

3. 债市资金面
   - 债市更关注资金成本、杠杆融资成本和利率定价。
   - 不能简单照搬股市资金面中的实体信用指标。
   - 应重点看DR007、R007、公开市场操作、国债收益率等。

4. 美国经济基本面
   - 不能只根据消费者信心判断需求降温。
   - 必须同时看非农、失业率、零售、ISM、CPI、PPI、核心PCE。
   - 若就业和消费仍有韧性，同时通胀上行，应表达为“增长仍有韧性，通胀压力上行”。

## 七、反向证据检查

在输出结论前必须检查反向证据。

常见情况：

1. 基本面偏弱，但PMI改善。
   - 不能写得过度悲观。
   - 应写“内需承压，但制造业景气改善”。

2. 资金价格边际上行，但公开市场回笼规模减少。
   - 不能直接写“资金面收紧”。
   - 应写“边际收敛”或“扰动有所缓和”。

3. A股估值偏高，但港股科技估值不高。
   - 不能笼统写“整体估值偏高”。
   - 应区分“A股偏高，港股相对中性”。

4. 债市基本面偏利多，但制造业景气改善。
   - 不能写“单边利多”。
   - 应写“中性偏多但利多边际弱化”。

## 八、第二轮展示规则

第二轮筛选是为了最终PPT展示。

原则：

1. 每个面最多展示1-3组关键指标。
2. 只展示最能解释结论的数据。
3. 数据必须能直接支撑判断词。
4. 括号里的数据要简洁，不堆砌。
5. 能用边际变化表达的，不优先使用容易造成误解的大同比数据。

例子：

股市基本面最终展示：

消费和投资偏弱（社零同比-0.6%、固投同比-4.1%、地产投资累计同比-16.2%）；制造业边际改善（6月制造业PMI为50.3、PMI新订单为51.2）；外需和盈利改善（出口较上月提升5.4个百分点、工业企业利润率较上月提升0.13个百分点）。

这里不是把所有基本面指标都写进去，而是选择最能解释“内需承压、结构支撑仍在”的几组数据。

## 九、最终结论句式

最终结论统一使用以下句式：

面：判断词：核心逻辑：分项解释（对应数据）；分项解释（对应数据）；分项解释（对应数据）。

其中：

- 判断词来自面级综合判断。
- 核心逻辑来自第一轮候选池。
- 括号数据来自第二轮展示指标。

示例：

股市基本面：中性：内需和地产仍承压，但制造业景气改善、外需与盈利修复提供结构性支撑：消费和投资偏弱（社零同比-0.6%、固投同比-4.1%、地产投资累计同比-16.2%）；制造业边际改善（6月制造业PMI为50.3、PMI新订单为51.2）；外需和盈利改善（出口较上月提升5.4个百分点、工业企业利润率较上月提升0.13个百分点）。

## 十、PPT样式规则

更新PPT时需要保持以下规则：

1. 不改变原PPT无关内容。
2. 只更新宏观结论部分。
3. 判断词加粗并标色。
4. 普通解释文字不加粗。
5. 上行箭头使用红色。
6. 下行箭头使用绿色。
7. “中性”偏灰色。
8. 偏积极、偏强、偏多等偏正面判断使用红色。
9. 偏高、偏空、偏谨慎等偏负面判断使用绿色。
10. 若文字放不下，先缩小字号，再考虑压缩文字。

## 十四、特别注意事项

1. 资金面不能轻易写“紧缩”。
   - 需要同时看价格、数量、政策利率、公开市场操作和阈值关系。

2. DR007必须和7天逆回购比较。
   - 高于政策利率说明资金边际收敛。
   - 但不等于直接进入紧缩状态。

3. SOFR必须和IORB比较。
   - SOFR低于IORB说明美元短端资金压力不明显。
   - SOFR高于IORB说明短端资金略有扰动。

4. 美国基本面不能只看单一软指标。
   - 必须覆盖就业、消费、景气、通胀。

5. 估值面必须给近三年分位。
   - 不能只写PE绝对值。

6. 北向资金成交额不是净流入。
   - 只能说明外资参与度或交易活跃度。

7. 股市和债市不能简单使用同一套指标。
   - 股市更关注盈利、风险偏好、流动性、估值。
   - 债市更关注资金成本、政策利率、收益率、供给压力和融资需求。

8. 最终PPT指标不是全部判断依据。
   - 面级判断来自第一轮候选池。
   - PPT展示来自第二轮筛选。
