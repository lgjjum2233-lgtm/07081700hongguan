# TREE 指标判断复核方法

后续判断指标结论时，不能只看当前值或单期环比，需要按以下顺序复核：

1. 当前状态：先判断当前值处于宽松、平稳、偏紧、扩张、收缩、改善或恶化的哪一侧。
2. 关键阈值：识别该指标真正重要的锚，例如 DR007 对比 7 天逆回购利率，PMI 对比 50，SOFR 对比 IORB，M1 对比 M2。
3. 跨阈值变化：如果指标从阈值一侧切到另一侧，优先级高于普通环比变化。比如 DR007 从低于 7 天逆回购转为高于 7 天逆回购，应识别为银行间资金面边际收紧。
4. 边际方向：结合环比、同比、同比增量判断边际是改善、走弱、收敛还是扩张。
5. 持续性：单期跨阈值应表述为“信号出现、需观察持续性”；连续维持则可升级为趋势判断。
6. 权重优先：面级结论优先引用权重高、变化大、跨阈值或能验证核心逻辑的指标，不逐项罗列。
7. 交叉验证：复杂指标必须结合相关指标判断，例如 DR007 结合 7 天逆回购，M1 结合 M2，财政存款结合财政支出，TGA 结合 ON RRP。
8. 数据一致性：先核对当前数据、数据日期、环比、同比、同比增量是否与 `指标判断` 或趋势解析一致；若文字判断滞后，以当前数据和日报底层复核为准。
9. 口径分层：日度和周度公开市场操作、成交额和净流入、估值绝对值和历史分位等口径不能互相替代；口径不清时标为待复核，不进入最终展示。

面级结论的标准表达结构：

总判断：状态 + 边际变化 + 主要矛盾。

子维度说明：选择权重大、变化大或跨阈值的指标，说明数据、阈值、边际变化和含义。
